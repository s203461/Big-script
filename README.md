# Wordle Bash
## Uruchomienie
```bash
./main.sh -f mail.txt -a 6 -g
```

## Opcje
- `-f` – plik słownika
- `-a` – liczba prób
- `-g` – tryb graficzny (Zenity)

## Pliki
- `main.sh` – główny skrypt
- `game.sh` – funkcje gry
- `config.sh` – obsługa konfiguracji
- `mail.txt` – słownik
- `config.txt` – zapis konfiguracji
- `wordle.1` – manpage
